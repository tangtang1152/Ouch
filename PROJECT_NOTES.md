# ouch Project Notes

## Project Name

ouch

## Core Problem

When receiving a massage, the user may feel pain in a position that changes quickly.  
It can be awkward or inefficient to keep verbally describing the exact position, especially when the masseur's hands are moving quickly.

## Initial Solution

Build a simple web page with:

- one large red **OUCH** button
- four direction buttons: up / down / left / right
- audio feedback for each button

This allows the user to communicate pain quickly with minimal speech.

## MVP Scope

### Must Have

- center OUCH button
- direction buttons
- sound playback
- basic mobile-friendly layout

### Not in V1

- gyroscope input
- motion detection
- volume key trigger
- user accounts
- feedback system
- donations
- app packaging
